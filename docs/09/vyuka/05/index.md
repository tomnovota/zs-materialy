# 05: Operační systém - prvky uživatelského rozhraní a uživatelské účty

!!! abstract "Cíl hodiny"
    Znát prvky operačního systému Windows a základních aplikací, a umět je ovládat pomocí klávesových zkratek.

---

## :material-web: Ovládání prohlížeče

### Práce s kartami

| Akce | Zkratka |
|------|---------|
| Další karta | ++ctrl+tab++ |
| Předchozí karta | ++ctrl+shift+tab++ |
| Nová karta | ++ctrl+t++ |
| Obnovit zavřenou kartu | ++ctrl+shift+t++ |
| Zavřít kartu | ++ctrl+w++ |

### Práce s okny

| Akce | Zkratka |
|------|---------|
| Nové okno | ++ctrl+n++ |
| Nové anonymní okno | ++ctrl+shift+n++ |

### Užitečné funkce

| Akce | Zkratka |
|------|---------|
| Stažené soubory | ++ctrl+j++ |
| Historie | ++ctrl+h++ |
| Záložky | ++ctrl+d++ |
| Aktualizovat stránku | ++f5++ nebo ++ctrl+r++ |

---

## :material-microsoft-windows: Ovládání operačního systému

!!! tip "Systémové možnosti"
    ++ctrl+alt+del++ – otevře nabídku s možnostmi (zamknout, odhlásit, správce úloh...)

### Přepínání oken

| Akce | Zkratka |
|------|---------|
| Další okno | ++alt+tab++ |
| Předchozí okno | ++alt+shift+tab++ |
| Přehled oken a ploch | ++win+tab++ |

!!! note "V přehledu oken (++win+tab++)"
    - Použij ++arrow-left++ ++arrow-right++ ++arrow-up++ ++arrow-down++ pro navigaci
    - ++delete++ zavře vybrané okno
    - Můžeš vytvořit novou virtuální plochu

### Umístění oken

| Akce | Zkratka |
|------|---------|
| Přichytit okno vlevo | ++win+arrow-left++ |
| Přichytit okno vpravo | ++win+arrow-right++ |
| Maximalizovat okno | ++win+arrow-up++ |
| Minimalizovat okno | ++win+arrow-down++ |

### Spouštění aplikací

| Akce | Zkratka |
|------|---------|
| Nabídka Start | ++win++ |
| Spustit příkaz | ++win+r++ |
| Průzkumník souborů | ++win+e++ |
| Nastavení | ++win+i++ |

!!! info "Další způsoby spuštění aplikací"
    - **Hlavní panel** – kliknutí na připnuté ikony
    - **Plocha** – dvojklik na zástupce

### Ovládání aplikací

| Akce | Zkratka |
|------|---------|
| Zavřít dialog/zrušit | ++esc++ |
| Zavřít aplikaci | ++alt+f4++ |
| Přepnout na celou obrazovku | ++f11++ |

## :material-file-document-edit: Ovládání textového editoru

### Pohyb v textu

| Akce | Zkratka |
|------|---------|
| Po znacích | ++arrow-left++ ++arrow-right++ |
| Po slovech | ++ctrl+arrow-left++ ++ctrl+arrow-right++ |
| Na začátek řádku | ++home++ |
| Na konec řádku | ++end++ |
| Na začátek dokumentu | ++ctrl+home++ |
| Na konec dokumentu | ++ctrl+end++ |
| O stránku výš | ++page-up++ |
| O stránku níž | ++page-down++ |

### Označování textu

!!! tip "Označování"
    Drž ++shift++ a použij klávesy pro pohyb – text se označí.

| Akce | Zkratka |
|------|---------|
| Označit vše | ++ctrl+a++ |
| Označit slovo | ++ctrl+shift+arrow-right++ |
| Označit řádek | ++home++ pak ++shift+end++ |

### Práce se schránkou

| Akce | Zkratka |
|------|---------|
| Kopírovat | ++ctrl+c++ |
| Vyjmout | ++ctrl+x++ |
| Vložit | ++ctrl+v++ |
| Zpět (Undo) | ++ctrl+z++ |
| Znovu (Redo) | ++ctrl+y++ |

### Mazání textu

| Akce | Klávesa |
|------|---------|
| Smazat znak vlevo | ++backspace++ |
| Smazat znak vpravo | ++delete++ |
| Smazat slovo vlevo | ++ctrl+backspace++ |
| Smazat slovo vpravo | ++ctrl+delete++ |

### Speciální klávesy

| Klávesa | Funkce |
|---------|--------|
| ++caps-lock++ | Přepíná psaní velkých/malých písmen |
| ++num-lock++ | Přepíná numerickou klávesnici |
| ++insert++ | Přepíná režim vkládání/přepisování |

---

## :material-lightbulb: Tipy pro procvičení

!!! example "Vyzkoušej si"
    1. Otevři prohlížeč a vytvoř 3 nové karty pomocí ++ctrl+t++
    2. Přepínej mezi nimi pomocí ++ctrl+tab++
    3. Zavři jednu kartu ++ctrl+w++ a obnov ji ++ctrl+shift+t++
    4. Otevři Poznámkový blok a napiš větu
    5. Označ celý text ++ctrl+a++ a zkopíruj ho ++ctrl+c++
    6. Otevři nový dokument a vlož text ++ctrl+v++
