# 📁 Microsoft OneDrive

> Cloudové úložiště pro bezpečné ukládání a sdílení souborů

## 🌐 Co je OneDrive?

**OneDrive** je cloudová služba od Microsoftu, která umožňuje:
- **Ukládání souborů online** - přístup odkudkoli
- **Synchronizaci** - stejné soubory na všech zařízeních
- **Sdílení** - spolupráci s ostatními
- **Automatické zálohování** - ochrana před ztrátou dat

## 🚀 Přístup k OneDrive

### Způsoby přístupu
- **Web prohlížeč:** [onedrive.live.com](https://onedrive.live.com)
- **Windows:** ikona cloudu v systémové liště
- **Mobilní app:** OneDrive z obchodu s aplikacemi
- **Integrovaný:** přímo v Office aplikacích

### První přihlášení
1. **Otevřít** OneDrive v prohlížeči
2. **Přihlásit se** Microsoft účtem (školním nebo osobním)
3. **Povolit synchronizaci** s počítačem (doporučeno)
4. **Nastavit složky** pro synchronizaci

## 📤 Základní funkce OneDrive

### Nahrávání souborů

#### Způsoby nahrání
```
1. Drag & drop - přetáhnout z počítače
2. Tlačítko "Nahrát" - vybrat soubory/složky
3. Automatická synchronizace - z počítače
4. Kamera telefonu - přímé pořízení fotky do cloudu
```

#### Podporované formáty
- **Dokumenty:** Word, Excel, PowerPoint, PDF
- **Obrázky:** JPG, PNG, GIF, BMP
- **Videa:** MP4, AVI, MOV
- **Další:** ZIP, TXT, a většina běžných formátů

### Organizace souborů

#### Doporučená struktura složek pro studenty
```
📁 Škola
  📁 2024-2025
    📁 Matematika
    📁 Čeština  
    📁 Angličtina
    📁 Fyzika
  📁 Archiv (starší ročníky)
📁 Osobní
📁 Fotky
📁 Projekty
📁 Sdílené se mnou
```

#### Tipy pro organizaci
- **Jasné názvy** - bez diakritiky, bez mezer
- **Datumy** - ve formátu RRRR-MM-DD
- **Verze** - číslování verzí dokumentů (v1, v2, final)
- **Archivace** - pravidelné přesouvání starých souborů

## 🔄 Synchronizace s počítačem

### Nastavení synchronizace
1. **Stáhnout OneDrive aplikaci** pro Windows/Mac
2. **Přihlásit se** Microsoft účtem
3. **Vybrat složky** k synchronizaci
4. **OneDrive složka** - speciální složka na počítači

### Výhody synchronizace
- 💾 **Offline přístup** - soubory dostupné i bez internetu
- 🔄 **Automatické zálohování** - změny se ukládají do cloudu
- ⚡ **Rychlý přístup** - jako běžné soubory na disku
- 🔍 **Vyhledávání** - najít soubory přes Windows Search

### Files On-Demand
- **Úspora místa** - soubory v cloudu, ale viditelné na počítači
- **Stahování na vyžádání** - otevřením se automaticky stáhne
- **Označení pro offline** - důležité soubory vždy dostupné

## 🤝 Sdílení souborů a složek

### Postupy sdílení
1. **Pravý klik** na soubor/složku → Sdílet
2. **Nastavit oprávnění:**
   - 👁️ Pouze zobrazení
   - ✏️ Úpravy povoleny  
   - ⏰ Časové omezení
   - 🔐 Heslo pro přístup

### Typy sdílení
```
🔗 Odkaz - kdokoliv s linkem může přistupovat
👥 Konkrétní lidé - pouze vybraní uživatelé  
🏫 Organizace - jen lidé ze školy/firmy
```

### Bezpečné sdílení
- **Kontrola oprávnění** - kdo má přístup
- **Časové omezení** - automatické ukončení přístupu
- **Heslo** - další úroveň zabezpečení
- **Pravidelná kontrola** - odebrání nepotřebných přístupů

## 🏫 OneDrive pro školu

### Office 365 Education
- **Zdarma pro studenty** - často poskytuje škola
- **Více prostoru** - až 1TB místa (místo 5GB)
- **Pokročilé funkce** - týmové weby, pokročilé sdílení
- **Integrace s Teams** - přímé napojení na školní platformu

### Výhody školního OneDrive
- **Neomezená spolupráce** - s učiteli a spolužáky
- **Větší bezpečnost** - spravováno školou
- **Integrace s Office** - Word, Excel, PowerPoint online
- **Zálohování školní práce** - ochrana před ztrátou

## 👥 Spolupráce na dokumentech

### Real-time spolupráce
1. **Sdílet dokument** s oprávněními k úpravám
2. **Současná práce** více lidí najednou
3. **Barevné kurzory** - vidíte, kdo kde edituje
4. **Automatické ukládání** - změny se ukládají průběžně
5. **Historie verzí** - návrat k předchozím verzím

### Nástroje pro spolupráci
- 💬 **Komentáře** - poznámky k částem textu
- 📝 **Sledování změn** - vidět všechny úpravy
- ✅ **Úkoly** - přiřadit konkrétním lidem
- 🎯 **Označení** - @mention pro upozornění

### Organizace týmového projektu
```
📁 Projekt - Životní prostředí
  📄 Osnova.docx (struktura projektu)
  📄 Výzkum.docx (shromažďování informací)  
  📄 Text.docx (finální verze)
  📊 Prezentace.pptx
  📁 Zdroje (obrázky, články)
  📄 Rozdělení úkolů.xlsx
```

- 🔄 **Automatické zálohy** - fotky z telefonu do cloudu
- 📤 **Rychlé sdílení** - odeslat soubor z telefonu
- 💾 **Offline soubory** - označit pro použití bez internetu

### Užitečné funkce
- **Automatické nahrávání fotek** - záloha všech fotek
- **Skenování dokumentů** - převod papíru na PDF
- **Rychlý přístup** - často používané soubory
- **Sdílení polohy** - koordinace se spolužáky

## 💾 Správa prostoru a optimalizace

### Kontrola využitého místa
```
OneDrive: Settings → More settings → Storage
Zobrazí:
- Celkové dostupné místo
- Využité místo podle typů souborů
- Největší soubory
```

### Optimalizace prostoru
- 🗑️ **Pravidelný úklid** - smazat nepotřebné soubory
- 📁 **Archivace** - starší ročníky do samostatné složky
- 📸 **Komprese fotek** - snížení kvality u méně důležitých
- 💾 **Lokální pouze** - často nepoužívané soubory

### Limity a plány
- **Osobní účet:** 5GB zdarma
- **Školní účet:** až 1TB (závisí na škole)
- **Microsoft 365:** 1TB + pokročilé funkce
- **Rozšíření:** možnost dokoupení dalšího prostoru

## ⚠️ Časté problémy a řešení

### Synchronizační problémy
- **Restartovat OneDrive** - ikona v systémové liště → Quit → Restart
- **Zkontrolovat internet** - stabilní připojení  
- **Místo na disku** - dostatek volného místa
- **Aktualizace** - nejnovější verze aplikace

### Problémy s přístupem
- **Zkontrolovat přihlášení** - správný Microsoft účet
- **Browser cache** - vymazat dočasné soubory
- **Jiný prohlížeč** - zkusit Firefox, Chrome, Edge
- **Kontakt IT podpory** - škola nebo Microsoft podpora

### Ztracené soubory
- **Koš OneDrive** - smazané soubory lze obnovit
- **Historie verzí** - obnovit předchozí verzi souboru
- **Vyhledávání** - použít různá klíčová slova
- **Sdílené složky** - kontrola v sekci "Sdílené se mnou"

## 🔒 Bezpečnost a ochrana dat

### Základní bezpečnost
- **Silné heslo** - kombinace písmen, čísel a znaků
- **Dvoufaktorové ověření** - další úroveň zabezpečení
- **Pravidelná kontrola** - kdo má přístup k souborům
- **Aktualizace** - nejnovější verze aplikací

### Strategie zálohování (3-2-1)
- **3 kopie** důležitých dat
- **2 různé typy** úložišť (cloud + USB/disk)
- **1 kopie** mimo domov (OneDrive v cloudu)
